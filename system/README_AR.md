# 🎓 منظومة التتبع الدراسي الذكية (مسار)

مرحباً بك في منظومة التتبع الدراسي، وهي تطبيق ويب محلي متطور مصمم لمساعدة الأساتذة في تدبير نقط التلاميذ وعرضها بشكل تفاعلي واحترافي، خاصة أثناء لقاءات أولياء الأمور.

## ✨ المميزات الرئيسية
- **لوحة تحكم تفاعلية**: رسوم بيانية توضح تطور مستوى القسم والتلميذ عبر 5 مراحل.
- **استيراد ذكي**: دعم سحب وإسقاط ملفات Excel أو مجلدات كاملة، مع كشط تلقائي للبيانات.
- **تقارير آلية**: توليد تقرير وصفي لمستوى التلميذ بناءً على نتائجه.
- **بحث مباشر**: ميزة البحث السريع مع اقتراحات فورية للوصول لبيانات أي تلميذ.
- **ذاكرة ذكية**: يحفظ النظام حالتك الحالية (القسم المفتوح، التلميذ المختار) حتى عند تحديث الصفحة.
- **معالجة مرنة**: نظام متطور للتعامل مع ملفات Windows المحجوزة لتجنب أخطاء الاستيراد.

---

## 🚀 دليل التشغيل السريع

### 1. المتطلبات الأساسية
تأكد من تثبيت **Node.js** على جهازك.

### 2. تثبيت المكتبات (للمرة الأولى فقط)
افتح واجهة الأوامر (Terminal) في مجلد المشروع ونفذ الأمر التالي:
```bash
npm install
```
*ملاحظة لمستخدمي Windows:* إذا ظهر لك خطأ (Execution Policy)، استعمل الأمر التالي بدلاً منه:
```bash
npm.cmd install
```
أو قم بتفعيل الصلاحيات عبر الأمر: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

### 3. تشغيل النظام
لتشغيل الخادم المحلي، نفذ الأمر التالي:
```bash
node server.mjs
```

### 4. فتح التطبيق
افتح متصفحك (Chrome أو Edge) وانتقل إلى الرابط التالي:
`http://localhost:3000`

## 📂 كيفية استيراد البيانات
1. انتقل إلى قسم **"استيراد البيانات"** من القائمة الجانبية.
2. يمكنك:
   - سحب ملفات Excel (شبكات تفريغ النقط) وإفلاتها في المنطقة المخصصة.
   - أو اختيار **"مجلد كامل"** يحتوي على ملفاتك؛ سيقوم النظام بالبحث في المجلدات الفرعية تلقائياً.
3. راجع قائمة الملفات التي ظهرت، ثم اضغط على زر **"ابدأ عملية الاستيراد"**.
4. سيقوم النظام بتنظيم الملفات في مجلدات "المرحلة 1-5" وتحديث لوحة التحكم فوراً.

---

## 🛠️ ملاحظات تقنية
- **قاعدة البيانات**: يتم تخزين البيانات في ملف `database.json`.
- **تنظيم الملفات**: يتم نقل ملفات Excel المنظمة إلى مجلد `نقط`.
- **في حال تعذر الاستبدال**: إذا كان الملف مفتوحاً في Excel، سيقوم النظام بتحديث البيانات في الموقع (Dashboard) وسيظهر لك تنبيه أصفر، ويمكنك استبدال الملف يدوياً لاحقاً.
